Buttons

M3 uses variants for emphasis — not a color attribute.

Variants — high → low emphasis

Elevated adds a shadow to lift the button off a colored or image surface — use it when a filled button would blend into the background. Avoid using it as the sole primary action.

Semantic actions

M3 expresses emphasis through variant, not color. Primary = filled, cancel/secondary = outlined or text, destructive = error role.

Primary action — filled
Secondary / cancel — outlined or text
Destructive — text or outlined, label carries the meaning
